# Soneium Score Bonus 8

Бот автоматически выполняет бонусные задания кампании [Soneium Score Season 8](https://soneium.org/en/blog/soneium-score-season-08)

**Старт сезона:** 11 марта 2026 | **Длительность:** 28 дней

---

## Сколько стоит выполнение заданий

| Задание | Стоимость | Комментарий |
|---------|-----------|-------------|
| **Startale GM** | Бесплатно | Только комиссия сети (газ) — копейки |
| **Startale Passkey** | Бесплатно | Только газ |
| **Kami (пазл)** | ~$3 за сезон | $1 за каждую неделю × 3 недели |
| **NekoCat Check-in (GMeow)** | Бесплатно | Только газ |
| **NekoCat Food** | ~$0.25 | 5 покупок еды по $0.05 каждая |
| **Press A (Unique NFT)** | ~$0.5 в среднем | Зависит от удачи — можно чуть меньше или больше |

> Газ в сети Soneium очень дешёвый, его стоимость незначительна и в таблице не учтена.

## Что делает каждый модуль — простым языком

### Startale GM — «Отправить GM»

Каждый день на сайте [app.startale.com](https://app.startale.com) нужно нажать кнопку **Send GM**. За сезон надо нажать 5 раз.

**Что делает бот:**
1. Открывает браузер (как обычный пользователь, через AdsPower)
2. Заходит на сайт startale.com
3. Нажимает кнопку «Send GM»
4. Запоминает, когда можно нажать снова (кулдаун ~24 часа)
5. Закрывает браузер

Запускай раз в день или реже — бот сам проверит, прошло ли время.

### Set up Passkey or social recovery method

С прошлого сезона не засчитвыается поэтому нужно выполнить снова + новая реализация которая даже в случае ошибки даст возможность восстановить passkey доступ к аккаунту

**Нужно заранее подготовить:**
- Аккаунты почты [FirstMail](https://firstmail.ltd) — одна почта на кошелёк, формат `email:password` в файл `firstmail_accounts.txt` советую покупать почты **Legacy** стоят всего 0.0020€
/шт, есть оплата криптой. Почты FirstMail обязательны для работы модулей на UI!
- API ключ AdsPower

### Kami — «Собрать пазл недели»

Каждую неделю на сайте [kamiunlimited.com](https://kamiunlimited.com) можно купить кусочек пазла за ~$1. За сезон 3 недели = 3 кусочка = $3.

**Что делает бот:**
1. Открывает браузер
2. Заходит на сайт Kami под твоим аккаунтом (используется почта FirstMail)
3. Покупает кусочек пазла этой недели (транзакция в блокчейне)
4. Закрывает браузер

**Нужно заранее подготовить:**
- Аккаунты почты [FirstMail](https://firstmail.ltd) — одна почта на кошелёк, формат `email:password` в файл `firstmail_accounts.txt` советую покупать почты **Legacy** стоят всего 0.0020€
/шт, есть оплата криптой. Почты FirstMail обязательны для работы модулей на UI!
- API ключ AdsPower

### NekoCat GMeow — «Ежедневный чек-ин»

Это задание похоже на «отметиться» в приложении. Нужно 10 раз за сезон подписать транзакцию чек-ина. Никаких денег — только газ.

**Что делает бот:**
1. Подключается к блокчейну Soneium
2. Проверяет, можно ли сделать чек-ин сегодня
3. Если да — отправляет транзакцию `signGMeow`
4. Запоминает счётчик (нужно набрать 10/10 за сезон)

Запускай раз в день. Бот сам пропустит, если уже отметился.

---

### NekoCat Food — «Купить еду для кота»

Нужно 5 раз купить еду в магазине NekoCat. Каждая покупка стоит ~$0.05 (0.000025 ETH по текущему курсу). Итого ~$0.25.

**Что делает бот:**
1. Подключается к контракту Food Shop
2. Проверяет, сколько покупок уже сделано
3. Покупает еду (транзакция в блокчейне, ~$0.05 + газ)
4. Повторяет до 5 раз за сезон

### Press A — «Гача на Unique NFT»

Это самое сложное задание: нужно получить **Unique** NFT (редкость grade 3) через гача-механику. Бот делает это по принципу «Free Path» — без покупки дорогих токенов, только за газ + небольшие VRF-сборы.

**Как работает гача в игре:**
- Есть NFT разных уровней: Rug (мусор) → Ape → Hodler → Alpha → Degen → LFG → **Unique** (цель)
- NFT можно продать, получив тикеты (Stone, Shell, Gold)
- Тикеты используются для спинов, которые дают NFT уровнем выше

**Что делает бот (пошагово):**
1. Проверяет — может, Unique уже есть? Если есть — всё, задание выполнено.
2. Свапает немного USDSC на LIFI (рандом 0.25-1$)
3. Минтит пачку Rug NFT (200 штук) - пачками по 10 шт 20 траназкций (0.2$)
3. Продаёт их все — получает тикеты Stone/Shell/Gold
4. Крутит спины за тикеты — получает NFT выше уровнем
5. Продаёт их снова, копит тикеты на более дорогие спины
6. Повторяет цикл, пока не выпадет Unique

Весь процесс занимает от нескольких минут до часа. Средние затраты ~$0.5 результаты затрат записываются в quest_results.json. Удача влияет на итоговую сумму.

## Установка

### Шаг 1. Установи Python

Скачай Python 3.11+ с [python.org](https://www.python.org/downloads/). При установке обязательно отметь галочку **«Add Python to PATH»**.

Проверь в терминале:
```
python --version
```
Должно показать версию 3.11 или выше.

### Шаг 2. Создай виртуальное окружение

Открой папку с ботом в терминале и выполни:

```bash
python -m venv venv
```

Активируй его:
```bash
# Windows (PowerShell или CMD):
venv\Scripts\activate

# Windows (Git Bash):
source venv/Scripts/activate

# Linux / macOS:
source venv/bin/activate
```

В начале строки должно появиться `(venv)` — это значит, окружение активно.

### Шаг 3. Установи зависимости

```bash
pip install -r requirements.txt
playwright install chromium
```

## Настройка файлов

### keys.txt — твои кошельки

Открой файл `keys.txt` и впиши приватные ключи — по одному на строку:
```
0xabcd1234...  ← первый кошелёк
0xef567890...  ← второй кошелёк
```

> Приватный ключ — это 64 символа (hex). Можно писать с `0x` в начале или без.

### proxy.txt — прокси 

Если используешь прокси — по одному на строку, в том же порядке что и ключи:
```
host:port:user:pass ← прокси для первого кошелька
host:port:user:pass  ← прокси для второго кошелька
```

Если файла нет или он пуст — бот работает без прокси (крайне не рекомендуется)

### adspower_api_key.txt — для Startale GM и Kami

Создай файл `adspower_api_key.txt` и вставь туда API ключ AdsPower (одна строка):
```
твой_api_ключ_здесь
```

### firstmail_accounts.txt — почты для Startale и Kami

Нужно для модуля **Kami**. Одна почта на кошелёк, в том же порядке что и ключи:
```
firstmail.ltd:password1
firstmail.ltd:password2
```

**ВАЖНО: каждому кошельку присваиваем свою почту, кошельки и почты размещаем 1к1 (это нужно для корректной работы модулей Startale и Kami, рандомизация в софте все равно присутствует и модули будут выполнятся рандомно)**

### config.toml — задержки и настройки

Открой `config.toml`. Основные параметры, которые стоит проверить:

```toml
delay_min = 30    # минимальная пауза между кошельками (секунды)
delay_max = 120   # максимальная пауза (секунды)
```

**Press A — режим только check-in (накопление наград):**

```toml
# по умолчанию выключено; чтобы включить — выставь true
press_a_checkin_only = false
```

- Если `press_a_checkin_only = true`, модуль `Press A` делает **только** ежедневный `checkIn()` и завершает работу (без bootstrap/spin/sell).
- Если **Unique уже засчитан на портале**, то check-in **не выполняется**.

Остальное можно не трогать — настройки по умолчанию подходят для большинства случаев.

## Как выглядит таблица прогресса

```
#    Адрес                                        GM     Passkey  Kami w1  Kami w2  Kami w3  GMeow    Food   PressA
-----------------------------------------------------------------------------------------------------------------------
1    0x1234...5678                                3/5    ok       ok       -        -        8/10     3/5    -
```

| Колонка | Что значит |
|---------|------------|
| GM | Прогресс отправки GM (3/5 = сделано 3 из 5) |
| Passkey | ok = passkey создан, - = нет |
| Kami w1/w2/w3 | ok = купил пазл этой недели, - = нет |
| GMeow | Прогресс чек-инов NekoCat (8/10) |
| Food | Прогресс покупок еды (3/5) |
| PressA | ok = Unique NFT получен, - = нет |

## Как работает защита от обнаружения (антисибил)

Чтобы аккаунты не были похожи на ботов, бот:

- **Перемешивает порядок задач** — не делает одно и то же на всех кошельках одновременно
- **Делает случайные паузы** между действиями (30–120 секунд между кошельками)
- **Добавляет случайность к газу** — каждая транзакция немного отличается
- **Использует разные прокси** — каждый кошелёк выходит в интернет со своего IP

## Если что-то пошло не так

**Бот пишет «квест уже выполнен, пропуск»** — всё нормально, бот проверил портал и увидел, что задание уже засчитано.

**Ошибка «keys.txt не найден»** — создай файл `keys.txt` в папке с ботом и добавь туда приватные ключи.

**Ошибка «adspower_api_key.txt»** — нужен API ключ AdsPower для Startale GM и Kami.

**Press A долго работает** — это нормально, гача-цикл может занять 10–60 минут в зависимости от удачи.
